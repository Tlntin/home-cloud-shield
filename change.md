# Changelog / 更新日志

## 0.0.2 - 2026-04-18

### 中文
- 新增 DNS 缓存时间设置，可按需缓存上游解析结果，减少重复请求。
- 新增自定义放行规则和自定义拦截规则，添加、管理规则更方便。
- 将缓存和日志保留相关选项迁移到独立的“设置”页面，配置结构更清晰。
- 优化设置页导航和输入区域布局，页面操作更顺手。
- 在主页、配置、我的三个主页面增加二次返回退出提示，减少误触直接退出应用的情况。

### English
- Added DNS cache duration settings to reduce repeated upstream lookups.
- Added custom allow and block rules for easier rule management.
- Moved cache and log retention options into a dedicated Settings page for a clearer configuration flow.
- Improved Settings page navigation and input layout for smoother operation.
- Added a double-back exit prompt on the Home, Config, and Me main pages to reduce accidental exits.
